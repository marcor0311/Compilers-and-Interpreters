# README - Tutorial

En archivo se debe sustituir por el nombre del archivo que se quiere ejecutar

## Para correr el proyecto
```
javac -d out *.java
java -cp out App archivo

Cada vez que se ejecuta, se generan dos archivos, TablaDeSimbolos.txt y tokens.txt

```
## Para la Fase Sintáctica

A continuación, se detallan los comandos específicos para la fase sintáctica. Cada comando ejecuta la aplicación con un archivo de prueba.


**Ejecutar con `prueba1.txt`**

```
    java -cp out App prueba1.txt
```

**Ejecutar con `prueba2.txt`**

```
    java -cp out App prueba2.txt
```

**Ejecutar con `prueba3.txt`**

```
    java -cp out App prueba3.txt
```

**Ejecutar con `prueba4.txt`**

```
    java -cp out App prueba4.txt
```

**Ejecutar con `errores.txt`**

```
    java -cp out App errores.txt
```
